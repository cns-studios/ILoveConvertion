2. Service Description

ILoveConvertion provides free file processing services including:

  • Image conversion and compression (JPEG, PNG, WebP, AVIF, HEIF, etc.)
  • AI background removal (using Silueta model)
  • PDF optimization (compression, linearization)
  • Audio conversion (MP3, FLAC, WAV, OGG, etc.)
  • Video compression (MP4, WebM, MKV)

Technical Specifications:

  • Maximum file size: 500MB (can vary based on server load)
  • Supported formats: As documented in README.md
  • Processing time: Varies by file size and complexity
  • File retention: 24 hours maximum, then automatic deletion
